Цель работы

Познакомиться с языком сценариев JavaScript, добавить на страницу «О себе» интерактивную функцию комментариев.

Ход Работы

Для добавления функционала комментариев, использовались следующие технологии:

  • JavaScript для добавления комментариев без перезагрузки страницы через манипуляцию DOM;
  • Python с FastAPI для сохранения комментариев на сервере в текстовом файле;

Сначала комментарии сохраняются в текстовом файле 'comments.txt'на сервере, а затем загружаются на страницу при ее загрузке. Это обеспечивает доступность всех добавленных комментариев даже после обновления страницы.
